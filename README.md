# CodeAlpha AI ChatBot

## Project Description
This is a simple AI ChatBot developed in Java using Java Swing. It provides a graphical user interface (GUI) where users can type messages and receive predefined responses.

## Features
- User-friendly GUI
- Interactive chat
- Responds to greetings like "Hi" and "Hello"
- Answers common questions
- Displays current date and time
- Tells a simple joke
- Help command to show available questions
- Exit using the "bye" command

## Technologies Used
- Java
- Java Swing (GUI)
- VS Code

## How to Run
1. Open the project in VS Code.
2. Compile the program.
3. Run `ChatBotGUI.java`.
4. Type your message in the text box and click **Send**.

## Sample Questions
- Hi
- Hello
- How are you?
- What is Java?
- What is AI?
- What is OOPS?
- What is your name?
- Who created you?
- Date
- Time
- Tell me a joke
- Help
- Bye
